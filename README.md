# Ripdiru

ruby gem to rip Radiru\*Radiru, NHK netradio

## Installation

Add this line to your application's Gemfile:

    gem 'ripdiru'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ripdiru
